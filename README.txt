### instructions to run the Synchronized code ####
1.unzip the folder in a directory
2.To check the synchronized version go to the Synchronized folder.
2.open a terminal in the Synchronized folder.
3.type "make" 
4.type "./output n" 
(where n is the no of threads you want to run)
5.type "make clean" to rebuild the program

### instructions to run the Unsynchronized code ####
2.Go to the Unsynchronized folder.
2.open a terminal in that folder.
3.type "make" 
4.type "./output n" 
(where n is the no of threads you want to run)
5.type "make clean" to rebuild the program
